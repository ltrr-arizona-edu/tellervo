"t_score_algorithm.png"		$$s_{xy} &=& \Sum x_i y_i - N (x_i - x_\mbox{avg}) (y_i - y_\mbox{avg}) \\
				s_{xx} &=& \Sum x_i^2 - N (x_i - x_\mbox{avg})^2 \\
				s_{yy} &=& \Sum y_i^2 - N (y_i - y_\mbox{avg})^2 \\
				r &=& s_{xy} / \sqrt{s_xx s_yy} \\
				t &=& r \sqrt{\frac{N-2}{1-r^2}}$$
